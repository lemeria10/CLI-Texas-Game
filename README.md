# 🃏 CLI Texas Hold'em Poker

A simple **Command Line Texas Hold'em Poker game** built with **Python** using **Object-Oriented Programming (OOP)**.  
The game allows a human player to play against a computer in the terminal.

## 🚀 Features
- 52-card shuffled deck
- Human vs Computer gameplay
- Hole cards and community cards (Flop, Turn, River)
- Simple betting system (Check, Call, Fold)
- Pot tracking and winner determination
- Replayable rounds

## 📂 Project Structure
poker-game/

## ▶️ Run the Game
